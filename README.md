# Diapo-voiture

Site web de présentation de montres de prestige (LuxeWatch).

## Description

Site web élégant présentant une collection de montres de prestige avec animations et design moderne.

## Fichiers

- `index.html` - Page principale du site
- `styles.css` - Styles CSS
- `script.js` - Scripts JavaScript
- `start-server.bat` - Script pour démarrer le serveur local (Windows)
- `start-server.ps1` - Script PowerShell pour le serveur HTTP
- `DEMARRER-LE-SITE.bat` - Script de démarrage alternatif
- `ouvrir-site.bat` - Script pour ouvrir le site dans le navigateur

## Installation et utilisation

### Démarrer le serveur local

Double-cliquez sur `DEMARRER-LE-SITE.bat` ou `start-server.bat` pour démarrer un serveur web local sur le port 8000.

Le site sera accessible à l'adresse : http://localhost:8000

## Technologies utilisées

- HTML5
- CSS3
- JavaScript (Vanilla)
- PowerShell (serveur HTTP)
